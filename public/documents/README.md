# Documents Klasörü

Bu klasöre PDF belgeleri yerleştirilebilir.

## Kullanım

- `/documents/privacy.pdf` → Bu klasördeki `privacy.pdf` dosyasına erişir
- `/documents/terms.pdf` → Bu klasördeki `terms.pdf` dosyasına erişir

## Not

Şu an bu klasör boş. PDF'ler yerine sayfa olarak `/documents/privacy` ve `/documents/terms` route'ları kullanılıyor.
